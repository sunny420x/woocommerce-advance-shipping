# จัดการค่าบริการบริษัทขนส่ง
จัดการค่าบริการบริษัทขนส่งสำหรับ Weight Based Shipping for WooCommerce รองรับการเลือกบริษัทขนส่งเองโดยลูกค้า รองรับการคิดค่าบริการขนส่งแบบขั้นบันใดของไปรษณีย์ไทย รองรับการกำหนดค่าขนส่งแบบคงที่จะสำหรับหมวดหมู่ของสินค้าบางหมวดหมู่

## วิธีติดตั้งปลั้กอิน ##
ไปที่ <code>/wp-admin/plugin-install.php</code> และอัปโหลดไฟล์ zip ของปลั้กอิน จากนั้นสามารถ <b>Activate</b> แล้วเริ่มใช้งานปลั้กอินจากเมนูได้เลย !
